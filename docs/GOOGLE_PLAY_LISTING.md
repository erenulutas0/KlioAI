# KlioAI Google Play Listing

This document keeps the Play Store copy, keyword direction, and screenshot plan in one place so the listing can be updated without searching through chat history.

## Global English Listing

### Short Description

AI-guided English speaking, pronunciation, writing, vocabulary, and daily review.

### Full Description

KlioAI helps English learners build a daily practice habit with AI-supported speaking, pronunciation, writing, vocabulary, and review.

Practice real conversations, write with structured feedback, save words with example sentences, and return to focused review sessions that keep difficult words visible. KlioAI is built for learners who want more than passive flashcards: each session is designed to turn saved vocabulary into active English use.

What you can do with KlioAI:

- Practice English speaking with AI-guided scenarios.
- Read short texts aloud and get a pronunciation clarity report.
- Improve writing with level-aware prompts and feedback.
- Save vocabulary and example sentences in one place.
- Generate translation practice from your own saved words.
- Learn daily words selected for useful English contexts.
- Review words with classic and neural review modes.
- Track progress, streaks, XP, and daily learning momentum.
- Use theme-based learning spaces that keep practice visually focused.

KlioAI focuses on English as the practice language. The learning profile can adapt AI explanations and translation practice to the learner's native/source language.

KlioAI özellikle İngilizce konuşma, kelime kullanımı ve günlük pratik alışkanlığı kazanmak isteyen öğrenciler için tasarlanmıştır.

## Spanish Experiment Listing

Use this only after Spanish source-language smoke testing passes in-app.

### Short Description

Practice English with AI speaking, pronunciation, writing, vocabulary, and review.

### Full Description

KlioAI is an AI-supported English learning app for learners who want to practice actively, not only memorize words.

You can speak with AI scenarios, read short texts aloud and receive a pronunciation clarity report, save vocabulary with example sentences, generate translation practice from saved words, and build a daily review habit with XP and progress tracking.

The app focuses on English practice. In Learning Profile, Spanish-speaking learners can choose Spanish as their native/source language so AI explanations and translation support can better match their learning context.

Main features:

- AI conversation practice for real-life English situations.
- Pronunciation practice with read-aloud reports.
- Writing prompts with AI-supported feedback.
- Vocabulary saving with example sentences.
- Translation practice from your own saved words.
- Daily words, XP, streaks, and progress tracking.

This experiment should measure whether Spanish-speaking users activate through pronunciation, speaking, or vocabulary-first flows.
